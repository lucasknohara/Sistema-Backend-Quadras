# Sistema de Gerenciamento de Quadras Esportivas

Projeto desenvolvido para prática de desenvolvimento backend com Node.js.

## Tecnologias utilizadas
- Node.js
- Express
- MySQL
- Arquitetura MVC

## Funcionalidades
- CRUD de quadras
- Filtros no backend
- Integração com banco de dados relacional

## Objetivo
Praticar desenvolvimento de APIs REST e organização de projeto utilizando boas práticas.
